Detailed Description

template<typename T, size_t N = 256>
class SM::SlotMap< T, N >

A data structure for managing objects using a SlotMap mechanism.

Template Parameters
TThe type of objects to be stored in the SlotMap.
NThe chunk size for allocating objects (default is 256).
